# DeepOrbit Graph Guide

Graph View is a navigation and knowledge-health surface, not the source of truth.

Recommended groups:

- `path:"20_Projects"` — active work
- `path:"30_Research"` — evidence and synthesis
- `path:"40_Wiki"` — durable concepts
- `path:"10_Diary"` — time-based context
- `tag:#task` — notes containing tasks

Enable arrows when reviewing knowledge flow. Hide attachments for conceptual reviews. Use Local Graph at depth 2 while writing, and periodically inspect global orphans outside `00_Inbox`.
