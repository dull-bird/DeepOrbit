# Project Rules

## README Sync Rule
When modifying `README.md` or `README_CN.md`, you **MUST** update both files to keep them in sync. Any content change to one must be reflected in the other. `README.md` is the English version and `README_CN.md` is the Chinese version.

## Skill & Command Sync Rule
When adding, removing, or modifying any skill (in `skills/`) or command (in `commands/`), you **MUST** check and update `README.md` and `README_CN.md` to reflect the change. This includes updating skill counts, mindmap diagrams, and the quick reference table.
